\# War Era Hub Extension



Browser extension for WarEraHub.



This extension syncs local WarEra browser data into WarEraHub, including profile, inventory resources, market prices, and company information.



\## What it does



\- Reads WarEra data already visible in the user's browser session.

\- Stores synced data locally in the browser.

\- Sends data to WarEraHub through local browser storage/events.

\- Does not store user cookies, JWTs, passwords, or tokens on any external server.



\## Install locally



1\. Download or clone this repository.

2\. Open Chrome.

3\. Go to `chrome://extensions`.

4\. Enable Developer mode.

5\. Click `Load unpacked`.

6\. Select this extension folder.

7\. Open WarEra and WarEraHub.
